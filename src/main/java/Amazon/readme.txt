时间复杂度的练习：
    通过O(n)的时间，把n的问题，变为了两个n/2的问题，时间复杂度是O(nlogn)
    通过O(1)的时间，把n的问题，变为了两个n/2的问题，时间复杂度是O(logn)

树的高度：
    平衡二叉树的高度是：logN
    其他二叉树的高度是：O(h)
